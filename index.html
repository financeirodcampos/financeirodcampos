Para publicar no GitHub, o passo a passo é: **1. Acesse o repositório** - Entre em **github.com/financeirodcampos/financeirodcampos** **2. Substitua o arquivo** - Clique no arquivo `index.html` - Clique no ícone de lápis ✏️ (Edit this file) no canto superior direito - Seleciona tudo (`Ctrl+A`) e apaga - Abre o arquivo novo aqui no chat, copia tudo (`Ctrl+A` → `Ctrl+C`) - Cola no GitHub (`Ctrl+V`) **3. Salve** - Role até o final da página - Clique em **"Commit changes"** - Clique em **"Commit changes"** novamente na janela que abrir **4. Aguarde ~1 minuto** e acesse o site normalmente --- Mas antes disso, você também precisa rodar os SQLs no Supabase para as novas tabelas funcionarem. Vai em **supabase.com → seu projeto → SQL Editor** e roda: ```sql -- Tabela Conciliações create table if not exists conciliacoes ( id uuid default gen_random_uuid() primary key, chave text unique not null, status text, obs text, created_at timestamptz default now() ); alter table conciliacoes enable row level security; create policy "allow all conciliacoes" on conciliacoes as permissive for all to anon, authenticated using (true) with check (true); -- Tabela DEMP/EMP (nova estrutura) drop table if exists demp_emp; create table demp_emp ( id uuid default gen_random_uuid() primary key, mes text not null, obra text, demp text, emp text, sienge_demp text, sienge_emp text, situacao text, obs text, created_at timestamptz default now() ); alter table demp_emp enable row level security; create policy "allow all demp_emp" on demp_emp as permissive for all to anon, authenticated using (true) with check (true); ```
